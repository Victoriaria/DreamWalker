## 标准走查路径（约 3–5 分钟）

**主路径：**

```
首页 → 输入旅人名 → 捕捉梦境碎片 → 生成梦之书
     → 巡梦回放 → 最后一页留言 → 梦之书架 / 日历回看
```

**操作步骤：**

1. 打开首页，输入旅人名（如「夜航者」）
2. 进入「捕捉」，输入 2–3 个梦境碎片
3. 点击「快捷语音」或语音按钮，展示非文本录入
4. 选择情绪、强度、身体感受、氛围色
5. 点击「生成梦之书」，进入巡梦回放
6. 展示分镜、视觉、输入线索；可开启声景与自动播放
7. 到最后一幕，填写并保存梦后留言
8. 进入梦之书架与日历，展示历史梦境
9. 刷新页面，证明数据仍存在

**推荐测试输入：**

```text
我在一座会呼吸的图书馆里醒来，书架像潮水一样退开，妈妈站在月光尽头没有说话。
后来我坐上一列没有车窗的列车，胸口发紧，车票上写着未来两个字，脚下却全是海水。
我想追上她，可走廊越来越长，灯一盏一盏熄灭，只剩海浪声在背后。
```

推荐标记：情绪「思念」· 强度 7/10 · 身体感受「胸口发紧、脚步沉」· 氛围色「青蓝」
