# SDD 验收清单 · 巡梦舱 DreamWalker V0.1

> **范围：** PRD §3.1 · `docs/PRD.md` · `app.js`  
> **用法：** 功能项可手工走查；Golden 场景见 `eval/scenarios.json`；录屏 Harness 见 `demo/`  
> **线上环境：** https://dreamwalker.vercel.app/

---

## 1. MVP 功能（PRD §3.1）

### 1.1 入口与隔离

- [x] 首页可输入旅人名并进入独立梦境空间
- [x] 不同旅人名使用不同 `localStorage` 键，数据互不混淆
- [x] 未设置旅人名时，非首页路由引导回首页

### 1.2 梦境捕捉

- [x] 支持多个梦境碎片输入
- [x] 支持 Web Speech API 语音输入
- [x] 语音受限时提供「快捷语音」片段兜底（非文本录入）
- [x] 可选择情绪、强度、身体感受、氛围色
- [x] 空输入时提示至少保留一个碎片

### 1.3 生成与回放

- [x] 点击「生成梦之书」后产出标题与 4–6 幕分镜
- [x] 分镜保留用户输入中的人物、地点、意象、身体感受
- [x] 巡梦回放展示分镜、视觉、输入线索、情绪标签
- [x] 支持自动播放与手动切换
- [x] 声景默认静音，用户可主动开启

### 1.4 情绪承接

- [x] 最后一幕提供梦后留言输入
- [x] 留言保存后可从书架查看

### 1.5 梦境管理

- [x] 梦之书架展示历史梦境，支持筛选与重新打开
- [x] 日历按日期展示梦境，可从日期进入详情或回放
- [x] 刷新页面后数据仍存在（localStorage 持久化）

---

## 2. SDD / 产品规范

### 2.1 端到端链路

- [x] 输入 → 生成 → 播放 → 留言 → 书架/日历，全链路可演示
- [x] 无后端依赖，静态部署可运行

### 2.2 视觉与宇宙映射

- [x] 记忆鲸落 / 月亮图书馆 两套视觉宇宙
- [x] 图片缺失时 Canvas 动态氛围兜底
- [x] 移动端单列布局与文字换行保护

### 2.3 文案与体验

- [x] 品牌名「巡梦舱 DreamWalker」一致
- [x] 空状态（无历史梦）有引导文案
- [x] 错误态（存储失败、语音失败）有可理解提示
