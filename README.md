# 巡梦舱 DreamWalker

## 100 字以内产品介绍

巡梦舱 DreamWalker 是一款梦境记录与沉浸式回看 Web 应用。用户写下梦境碎片后，系统生成梦之书、分镜视觉与声景，并通过日历、书架和最后一页留言安放情绪。

Slogan：每个梦，都值得被再次走过。

## 项目说明

巡梦舱不是普通记事本，而是一个帮助用户保存梦、重新进入梦、并承接梦后情绪的空间。用户可以把醒来后残留的碎片写下来，补充情绪、身体感受和氛围色；系统会提取梦里的意象、人物、地点与身体感，生成一本可回放的梦之书。

产品围绕赛题「帮助用户记录、管理梦境，并沉浸式承接用户的情绪」设计，主链路为：

1. 输入旅人名，进入独立梦境空间。
2. 记录多个梦境碎片，可使用文字、语音输入或快捷语音片段。
3. 标记核心情绪、强度、身体感受和氛围色。
4. 生成梦之书，得到标题、梦境理解、4-6 幕叙事和视觉场景。
5. 进入巡梦回放，查看分镜、动态氛围、输入线索和声景。
6. 在最后一页留下情绪留言。
7. 通过梦之书架和日历回看历史梦境。

## 核心功能

- 梦境记录：支持梦境名、多个梦境碎片、情绪、强度、身体感受、氛围色。
- 非文本录入：支持 Web Speech API 语音捕捉；浏览器限制麦克风时，可用快捷语音片段继续演示。
- 梦境理解：提取关键意象、人物、地点、身体感受和梦境色彩。
- 叙事生成：根据输入碎片生成 4-6 幕梦境分镜，保留用户输入中的关键元素。
- 视觉体验：使用本地产品素材、Canvas 动态氛围和输入线索浮层呈现梦境。
- 声景回放：支持手动开启 Web Audio 声景，配合自动播放和分镜进度。
- 情绪安放：最后一幕根据梦境世界给出反馈问题，并保存梦后留言。
- 梦境管理：梦之书架支持历史梦境浏览、筛选、详情和重新打开。
- 日历视图：按日期显示当天梦境，可从日历进入详情或回放。
- 数据持久化：使用 localStorage 保存数据，刷新或重新打开后仍可找回。
- 用户隔离：不同旅人名对应不同本地存储键，梦境数据互不混淆。

## 运行说明

本项目为纯前端 Web 应用，无需安装依赖。

推荐运行方式：

```bash
cd DreamWalker
python3 -m http.server 5173
```

然后在浏览器访问：

```text
http://localhost:5173
```

也可以直接打开：

```text
index.html
```

评审或录屏时更推荐使用本地静态服务方式，语音、图片和刷新回看流程更稳定。

## 演示路径

建议演示视频控制在 3-6 分钟，按下面顺序录制：

1. 打开首页，输入旅人名，说明不同旅人拥有独立梦境空间。
2. 进入「捕捉」，输入 2-3 个梦境碎片。
3. 点击「快捷语音」或演示语音输入，证明支持非文本录入。
4. 选择情绪、强度、身体感受和氛围色。
5. 点击「生成梦之书」，进入巡梦回放。
6. 展示分镜标题、原始输入线索、动态视觉、情绪标签和自动播放。
7. 开启声景，说明声景默认静音、用户可主动打开。
8. 到最后一幕，填写并保存梦后留言。
9. 返回梦之书架，展示历史梦境和留言。
10. 进入日历，点击有梦境的日期，从日历打开详情或重新回放。
11. 刷新页面，展示数据仍然存在。

## 成果展示材料

### 展示标题

巡梦舱 DreamWalker：每个梦，都值得被再次走过。

### 展示摘要

巡梦舱把梦境碎片、心境标记、视觉分镜、声景和最后一页留言串成一条完整体验。它帮助用户在醒来后留住梦，也在回看梦时给出温柔的情绪承接。

### 评审重点对应

- 端到端链路：输入碎片后生成梦之书，并进入可观看、可播放、可留言的回放体验。
- 碎片理解：生成结果保留人物、地点、意象、身体感受和情绪。
- 多模态体验：包含产品视觉素材、Canvas 动态氛围、声景、分镜回放和日历回看。

## 数据说明

当前旅人名保存在：

```text
dreamwalker:currentTraveler
```

梦境数据按旅人名隔离保存在：

```text
dreamwalker:data:<traveler-name>
```

因此，同一浏览器中切换旅人名后，会进入不同的本地梦境空间。

项目不依赖后端数据库、登录系统或第三方框架，适合现场稳定演示。

## 可靠性与回退策略

- 空输入：提示用户至少留下一个梦境碎片。
- 超长输入：单个碎片限制长度，并提示保留前段内容。
- 语音不可用：继续支持文字输入与快捷语音。
- 语音失败：显示可重试提示，不阻塞生成流程。
- 图片缺失：自动回到本地 Canvas 梦境视觉。
- 外部插画未配置：主链路仍可完整生成、保存、回放和回看。
- 存储失败：提示本地空间不足，引导删除旧梦后重试。
- 无历史记录：书架和日历展示空状态并引导创建。
- 移动端：页面提供单列布局和文字换行保护。
